### Загальна динаміка виробництва

| Рік | Обсяг (млрд грн) | Зміна YoY |
|:---:|---:|:---:|
| 2013 | 2 593,3 | — |
| 2014 | 2 885,6 | +11,3% |
| 2015 | 3 449,9 | +19,6% |
| 2016 | 4 217,8 | +22,3% |
| 2017 | 5 329,0 | +26,3% |
| 2018 | 6 207,7 | +16,5% |
| 2019 | 6 981,9 | +12,5% |
| 2020 | 7 294,4 | +4,5% |
| 2021 | 9 639,3 | +32,1% |
| 2022 | 7 712,5 | **−20,0% ⚠️** |
| 2023 | 8 978,7 | +16,4% |
| 2024 | 9 388,0 | +4,6% |

**Ключові тенденції:**

- **2013–2021 — стале зростання:** середньорічний приріст ~15,7%, обсяг виріс у 3,7 рази. Основні чинники — девальвація гривні та розширення агро- й промислового виробництва.
- **2022 — шок повномасштабного вторгнення:** падіння на 20% (−1 927 млрд грн). Найбільше постраждали добувна промисловість, будівництво та металургія.
- **2023–2024 — відновлення:** обсяг 2024 р. (9 388 млрд грн) вперше перевищив пік 2021 р., що свідчить про адаптацію економіки до умов війни.

### Структура бізнесу (частка від загального обсягу)

| 2013 | 43,8% | 36,4% | 19,8% | 8,8% |
| 2017 | 38,3% | 37,0% | 24,7% | 13,3% |

## 🏗️ Архітектура

```
ukraine-dashboard/
├── Program.cs                  # ASP.NET Core Minimal API
├── backend.csproj              # .NET 8
├── data.json                   # 11 628 записів (спарсено з xlsx)
├── appsettings.json
└── frontend-src/               # Vite + React
    └── src/
        ├── App.jsx
        └── components/
            ├── Summary.jsx         # KPI + динаміка обсягів
            ├── SectorsChart.jsx    # Галузі: bar + trend charts
            └── KvedExplorer.jsx    # Провідник КВЕД + порівняння
```

**Принцип роботи:** при старті бекенд завантажує весь `data.json` у пам'ять як `List<EconomicRecord>` і реєструє як singleton. Це забезпечує відгук API < 5 мс без потреби у БД.

## 🔌 API Reference

| Метод | Ендпоінт | Опис |
|---|---|---|
| `GET` | `/api/summary` | Зведені дані по всіх роках (рядок `TOTAL`) |
| `GET` | `/api/sections` | Секції КВЕД верхнього рівня (A–S) з динамікою по роках |
| `GET` | `/api/kved/{code}` | Дані для конкретного коду КВЕД |
| `GET` | `/api/top?year=2023&limit=10` | Топ галузей за обсягом для вказаного року |
| `GET` | `/api/search?q=...` | Пошук по назві або коду КВЕД (до 20 результатів) |
| `GET` | `/api/kved-tree` | Повне дерево всіх 969 кодів КВЕД |

## 🧩 Компоненти фронтенду

| Компонент | Файл | Опис |
|---|---|---|
| **Summary** | `Summary.jsx` | KPI-картки (обсяг, YoY, частка бізнесу), графіки динаміки з перемиканням Area / Line / Bar |
